# Ai-prompt for learning

本项目是AI辅学的提示词。

主要有以下几个功能

- 课堂学习:引导式教学，AI依据老师的授课内容，先讲解，再向你提问，遇到你不懂的递归式解答
- 题目练习: 对于零基础的同学，快速补全必要的知识点，也是AI逐渐提问判断水平并做相应的讲解
- 课堂文件处理: 把原始的语音稿转化成更加可读的内容

### 经验

- AI授课的过程中，容易遇到AI仅仅在开头阅读一遍课程内容,后续不再阅读，在记忆中出现混乱，导致后续出现幻觉，虚构课程内容的问题。目前的方式是强制AI在每一轮交互中说明来源文件的小标题与对应PPT页码，供用户核对

### 设计思路

- AI并不完美，我们也不能期望AI完美，在调用工具与分析时会出现很多错误，我和他说不要跳过内容不讲有时候还是会跳过内容，这并不意味着我们不能用AI了，而是加以约束监督，在AI出错时不断提醒，多看着几眼，然后让AI为我们所用。我们可以设计更好的工作流来充分发挥AI的潜力。

  

- 借助AI学习，不能替代我们自己反复地复习，不能替代手动总结整理的过程，因为自己手动总结整理的时候本身就是知识内化过程，不能替代自己手动做题与输出，AI用于快速帮我们理解。可以让它柏拉图式教学，可以让它整理课程笔记帮助我们学习时候的加速。
